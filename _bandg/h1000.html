H1000

The H1000 system

Introduced in 2002?
All the instruments are based on the Fastnet bus, including the analogue displays.
There is no central computer to calculate extra information.
 

Some info on a H1000 CND Plotter:

It uses a VL2020 battery as a backup
Manual at B&G site: http://www.bandg.com/upload/BandG/Documents/Manuals/h1000_cnduser_english.pdf
 
It supports uploads of GPS Waypoints, but you have to be in the waypoint screen to make this work.
