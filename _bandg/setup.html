H2000 Hydra Basic Setup

DEPTH -> DEPTH ==> CALIBRATE -> DATUM
SPEED -> BOAT SPD ==> CALIBRATE -> MANL CAL -> SINGLE (or use AUTO CAL)
WIND -> MEAS W/A ==> CALIBRATE -> MHU ANGL   (or sometimes APP W/A)
 
NAVIGATE -> COURSE  ==> CALIBRATE -> CAL VAL1 -> HEADING NODE 
NAVIGATE -> D/R COURSE ==> CALIBRATE -> CAL VAL1 -> HEADING NODE
(5, 9, 15,16,17,18 or 96,97,...) heading node 9 is NMEA through Performance Processor instead of FFD
http://primus.bandg.com/icrbg/BandG%20FAQs/H2000 FAQS/FAQ - How do I change the Heading Node.pdf
The node numbers are entered base-10 (e.g. 96) but displayed base-16 (0x60)
 
NAVIGATE -> HEADING ==> CALIBRATE -> CAL VAL 1 -> HEADING OFFSET
MOTOR -> VOLTS ==> CALIBRATE -> CAL VAL 1
 
For better wind data, enter the correction table found in the X35 tuning guide http://www.skilux.fi/Link.aspx?id=1009319
WIND -> TRUE W/A ==> CALBRATE -> CORRECTION
WIND -> TRUE W/S ==> CALBRATE -> CORRECTION
 
Further, change the DAMPING of some functions, depending on the wind conditions

H2000 Pilot Setup

To enter setup, press both < and > next to COURSE (3rd row of buttons)
Enter all data (B TYPE=SAIL, H SRC=18, H OFF=0,DIP=2,DRIVE=A, S CAL=6.25,B LEN=12.0,B LAG=0.30,R GAN=2)
RUD P = rudder Port, RUD S = rudder Starboard, RUD M=mid and then select RUD T, this will determine the time from Port to Starboard
 
Sea Trial: Do a compass SWING, make a slow 360, later test the system by changing the boat direction by 90 or 120 degrees and determine boat lag.
 
During Sailing, select RESP and choose between ECON, NORMAL or DOWNWIND for more quicker response
http://www.bandg.com/upload/BandG/Documents/Manuals/newhpilot.pdf
http://www.bandg.com/upload/BandG/Documents/Div/hpilot.pdf (pre July 2004)

H2000 Performance Setup

NAVIGATE -> LEEWAY ==> CALIBRATE -> CAL VAL1  = Factor K (between 8 and 10 on a fast yacht?)
PERFORMANCE -> HEEL ==> CALIBRATE -> CAL VAL 1 -> OFFSET
PERFORMANCE -> TRIM ==> CALIBRATE -> CAL VAL 1 -> OFFSET
 
For entering your performance data, use a program like Deckman or Expedition, which will save you a lot of work.
